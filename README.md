# 📌 Task 4 – Image Section with Button

## 📖 Description

This project demonstrates a simple webpage layout that includes an image display section along with a user interaction button. The layout is designed using HTML and CSS with proper spacing using viewport units.

---

## 🚀 Features

* Responsive image display
* Interactive button
* Clean and minimal UI design
* Spacing managed using viewport units (`vh` and `vw`)
* No scrolling behavior without using `overflow: hidden`

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 📂 Project Structure

```
Task-4/
│── index.html
│── style.css
│── image/
│   └── image.jpg
```

---

## 💻 Implementation Details

### 🔹 HTML

* Created a container div with class `imageSection`
* Added an image inside the container
* Added a button for user interaction

### 🔹 CSS

* Used `margin: 10vh 25vw` to provide spacing:

  * 10% from top and bottom
  * 25% from left and right
* Set container height to `80vh` to avoid overflow
* Styled button with hover effect
* Ensured responsive image using `width: 100%`

---

## 📐 Layout Explanation

* Total vertical space = `10vh (top) + 80vh (content) + 10vh (bottom) = 100vh`
* This ensures the content fits perfectly within the viewport without scrolling
